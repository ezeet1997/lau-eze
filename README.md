# Centro Anti Aburrimiento

Página estática para subir a GitHub Pages.

## Archivos

- index.html
- styles.css
- script.js

## Cómo subir a GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir estos tres archivos.
3. Ir a Settings > Pages.
4. En Branch elegir `main` y carpeta `/root`.
5. Guardar.
6. GitHub te va a dar un link público.
